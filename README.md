🌐 Personal Web-Portfolio


Современное адаптивное веб-портфолио студента 3 курса TOP ACADEMY, созданное для демонстрации проектов и навыков веб-разработки.

🛠 Архитектура проекта


Проект реализован с разделением логики, стилей и контента для удобства поддержки и масштабирования:

index.html — Главная страница: профиль, технологический стек (HTML, JS, Go, C++, C#) и навигация.

projects.html — Каталог проектов: интерактивная сетка (Grid) с карточками выполненных работ.

details.html — Страница кейса: подробный разбор конкретного проекта с перечислением целей и стека.

style.css — Стилизация: использование CSS-переменных (:root), Flexbox и Grid для адаптивности.

scripts.js — Интерактивность: управление темами, анимации появления контента и обработка событий.

🚀 Функционал


Приложение включает в себя следующие возможности:

Theme Management: Динамическое переключение между светлой и темной темами оформления.

Responsive UI: Полная адаптация интерфейса под мобильные и десктопные устройства.

Smooth UX: Плавные переходы и анимация прозрачности при загрузке страниц.

Interactive Portfolio: Система карточек проектов с визуальным откликом при наведении.

Contact Integration: Заглушка для обратной связи с выводом контактных данных через уведомления.

💻 Технологический стек


Frontend: HTML5, CSS3 (Custom Properties, Flex, Grid).

Logic: JavaScript (ES6+) — DOM Manipulation, Event Listeners.

Other Skills: Опыт разработки на Golang, C++, C# и работа с Git/GitHub.

⚙️ Как запустить


Склонируйте репозиторий на локальную машину.

Убедитесь, что все файлы (.html, .css, .js, .jpg) находятся в одной корневой директории.

Откройте файл index.html в любом современном браузере.
