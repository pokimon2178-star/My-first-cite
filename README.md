🚀 Web Portfolio — Garibyan Georgiy
Это мой персональный проект-портфолио, разработанный для демонстрации навыков веб-разработки и других программных проектов.

🛠 Особенности интерфейса
Dynamic Theme Switching: Реализована кастомная смена цветовой схемы (темная/светлая) через CSS-переменные и JS.

Responsive Design: Сайт полностью адаптирован под мобильные устройства и десктопы благодаря Flexbox и Grid.

UX Animations: Плавное появление контента при загрузке страницы для лучшего пользовательского опыта.

💻 Мой Технологический Стек
Web: HTML5, CSS3, JavaScript (ES6+).

Системное программирование: Golang, C++, C#.

Инструменты: Git, GitHub.
