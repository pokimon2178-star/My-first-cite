📝 О проекте
Данный проект представляет собой персональную веб-страницу, созданную для демонстрации навыков программирования и портфолио выполненных работ. Сайт включает в себя разделы с информацией об авторе, списком проектов и детальным описанием конкретных кейсов.

Основные возможности:

Адаптивная верстка: Корректное отображение на различных устройствах (Flexbox/Grid).

Темная тема: Реализована возможность переключения между светлым и темным режимами оформления.

Плавная анимация: Использование JavaScript для создания эффекта появления контента при загрузке страницы.

🛠 Технологический стек
В ходе разработки использовались следующие инструменты:

Frontend: HTML5, CSS3.

Scripting: JavaScript (ES6+).

Верстка: Flexbox, CSS Grid.

Контроль версий: Git / GitHub.

Я также обладаю навыками работы с Golang, C++ и C#.

📂 Структура репозитория
index.html — Главная страница с информацией о себе и стеком технологий.

projects.html — Список реализованных проектов.

details.html — Подробное описание конкретного проекта.

style.css — Стилизация интерфейса и логика тем оформления.

scripts.js — Интерактивные элементы (переключатель темы, уведомления, анимации).
